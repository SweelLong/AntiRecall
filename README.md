# 防撤回插件

### 配置文件

-   "ClearHistoryByNum":      到达该数量就删除储存在文本文件里的历史消息                       

### 相关指令

- 撤回即触发

### 关于

- 这是基于KnifeHub - KonataPlugin的开源插件，功能是防止消息撤回。

- 倘若机器人频繁发送 “该部分已隐藏” 请重启KonataPlugin插件（建议重新登录机器人）

- 启用前需保证 KonataPlugin 处于已启用状态
- 
- 可自行下载源码并编译
